# Retail Sales Analysis

**Objective:** Analyze retail sales data, understand customer behavior, and generate business insights.

## Tools
- Python (Pandas, Matplotlib)
- CSV dataset
- Jupyter Notebook

## Features
- Data cleaning and preparation
- Revenue calculation
- Exploratory data analysis (trends, top products, revenue)
- Customer segmentation and insights

## Insights
- Top 10 products generate most revenue
- High-value customers are identified to aid marketing
- Revenue trends show seasonal patterns. Which month generates the most revenue

## Instructions
1. Clone the repository
2. Open the notebook in Jupyter or Google Colab
3. Run all cells
